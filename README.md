# 🔐 Password Manager - Менеджер паролей

**Автор:** Иван Иванов  
**Версия:** 2.0.0  
**Лицензия:** MIT

## 📝 Описание

Графическое приложение для безопасного хранения, генерации и управления паролями. Разработано с использованием библиотеки Tkinter.

### Возможности

- ✅ Добавление, редактирование и удаление записей
- 🔐 Просмотр сохранённых паролей
- 🎲 Генерация случайных паролей с настройками:
  - Длина от 8 до 32 символов
  - Заглавные и строчные буквы
  - Цифры
  - Специальные символы
- 💾 Автоматическое сохранение в JSON-файл
- 📋 Копирование паролей в буфер обмена
- 🎨 Тёмная тема интерфейса
- 🔍 Валидация входных данных

## 🚀 Запуск приложения
